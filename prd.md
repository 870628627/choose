# PRD: 家庭版 A 股研究小本自动数据版

## 1. 产品目标

把现有“家庭版 A 股研究小本”升级成自动获取数据版本。用户只输入股票代码，系统自动获取股票基础信息、估值指标、财务指标、公告信息，并基于数据库中的结构化数据和公告标题生成简单解释。

本工具只用于家庭自用的股票信息整理、研究对比和复盘，不做荐股，不做交易，不提供涨跌预测、目标价或自动买卖点。

## 2. 目标用户

- 家庭成员，例如我和爸爸。
- 使用场景是整理自选股票、理解公司基本情况、横向对比、记录判断、后续复盘。
- 用户不需要金融终端级实时数据，只需要稳定、可解释、可回看。

## 3. 技术范围

- 前端：React + Vite + TypeScript + Tailwind CSS。
- 后端：Node.js + Express。
- 数据库：SQLite。
- 数据采集服务：Python `data-worker`。
- 数据采集：优先 AKShare，预留 Tushare token 配置。
- AI 分析：预留 OpenAI / DeepSeek API 配置，MVP 使用模拟 AI 分析接口。
- 数据频率：第一版只支持日频行情、估值和财务数据，不支持分钟级行情和实时交易。

## 4. 核心功能

### 4.1 股票添加

用户输入股票代码，例如 `600519`、`000001`、`300750`。

系统自动识别：

- 市场。
- 股票名称。
- 行业。
- 公司基本信息。

保存到自选股票池。

### 4.2 自动数据同步

新增 Python `data-worker`，提供以下任务：

- `fetch_stock_basic`：获取股票基础信息。
- `fetch_daily_metrics`：获取每日估值和行情指标，包括收盘价、PE、PE_TTM、PB、PS、股息率、市值、换手率。
- `fetch_financials`：获取财务指标，包括营收、净利润、营收增速、净利润增速、毛利率、净利率、ROE、资产负债率、经营现金流。
- `fetch_announcements`：获取巨潮资讯公告列表，包括公告标题、公告时间、公告类型、公告链接。
- `sync_all`：一键同步所有自选股数据。

### 4.3 数据库表

SQLite 需要包含：

- `stocks`
- `daily_metrics`
- `financial_metrics`
- `announcements`
- `research_scores`
- `family_notes`
- `review_records`
- `sync_logs`

### 4.4 自动评分系统

评分名称统一为“研究评分”，总分 100 分：

- 行业前景：20 分。
- 公司竞争力：20 分。
- 财务质量：20 分。
- 估值合理性：20 分。
- 风险控制：20 分。

评分用于辅助研究和复盘，不代表买入、卖出或持有建议。

### 4.5 风险标签规则

系统根据结构化数据和公告标题自动打标签：

- 估值偏高
- 现金流风险
- 债务风险
- 业绩下滑
- ROE下降
- 减持风险
- 诉讼风险
- 监管风险
- 商誉风险
- 业绩变脸风险

### 4.6 AI 分析

AI 只基于数据库中的结构化数据和公告标题生成解释，不允许自由发挥。

AI 输出包括：

- 公司一句话介绍。
- 财务表现解释。
- 估值高低解释。
- 主要风险解释。
- 同行对比解释。
- 爸爸版通俗解释。

AI 输出禁止出现：

- 推荐买入
- 目标价
- 必涨
- 稳赚
- 抄底
- 牛股
- 荐股

### 4.7 页面

- 首页：自选股票列表。
- 股票详情页：自动数据 + AI 解释 + 我和爸爸的笔记。
- 股票对比页：支持 2-3 只股票横向对比。
- 风险排雷页：展示所有股票的风险标签。
- 复盘页：记录当时判断和后续结果。
- 同步管理页：手动点击“同步数据”，查看最近同步时间和错误日志。

### 4.8 自动同步

- MVP 支持手动点击同步。
- 预留每天 18:30 自动同步扩展点。
- 每次同步写入 `sync_logs`，记录成功、失败、错误原因。

## 5. 合规要求

页面底部固定显示：

> 本工具仅用于家庭自用的股票信息整理、研究对比和复盘，不构成任何投资建议。股市有风险，投资需谨慎。

全系统不出现荐股、预测涨跌、目标价、自动买卖点相关功能。

## 6. MVP 验收标准

- 用户可以启动前端、后端和 Python data-worker。
- 用户可以输入股票代码并添加到自选股票池。
- 系统能保存股票基础信息、日频估值行情、财务指标、公告、研究评分、风险标签、家庭笔记、复盘记录、同步日志。
- 没有 AKShare 环境时，系统能使用演示数据完成端到端流程。
- 前端包含首页、详情页、对比页、风险排雷页、复盘页、同步管理页。
- AI 分析接口可用，并明确是基于库内数据生成的模拟解释。
- README 包含安装、启动、配置和合规边界说明。
